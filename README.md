Mitakuuluu - Reborn
==========

Mitakuuluu - indie WhatsApp-compatible client for SailfishOS. Written in Qt5, Mitakuuluu based on Persecute client for Harmattan Qt4 project, which is based on Yappari client for Maemo5.
